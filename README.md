# README

## 開発言語
- Ruby 3.0.1
- Rails 6.1.6

## 就業Termの技術
- devise
- rails_admin
- cancancan
- carrierwave
- kaminari
- フォロー機能
- メッセージ機能

## カリキュラム外の技術
- Google map API
- geocoder
- ransack
- simple_calendar

## 実行手順
```
$git clone git@github.com:Kazuoki-Kinoshita/farm_link.git
$cd farm_link
$bundle install
$yarn install
$rails db:create && rails db:migrate
$rails s
```
